
Secure Patient Data Platform on Google Cloud (Capstone)

        Grade: A (93/100) | Distinguished Capstone Project
        Architected a secure, HIPAA-aligned cloud solution to address UN SDG Goal 3 (Good Health & Well-being), demonstrating how cloud-native security can protect sensitive patient health information (PHI) at scale.

        - Security Architecture: Designed a Zero Trust environment on Google Cloud Platform (GCP), implementing comprehensive risk analysis and operational mitigation strategies to prevent data exfiltration.
        - Risk & Compliance: Conducted a deep-dive technical analysis of cloud vulnerabilities, mapping controls to regulatory standards for data privacy and integrity.
        - Cost & Scale: Engineered the solution for high availability and cost-efficiency, balancing rigorous security controls with scalable infrastructure requirements.
        - Outcome: Awarded an "A" Grade for exceptional research, strong technical analysis, and innovative use of cloud security services.
